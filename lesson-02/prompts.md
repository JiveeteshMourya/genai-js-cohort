## Prompting

1. Zero Shot Prompting - Direct instructions dena
2. Few Shot Prompting - Zero Shot Prompting with examples outputs (direct instructions + examples outputs), it influences the output into betterment
3. Chain of Thought Prompting - giving instructions like "hey model have some breakdown of problem and then solve it before giving final output"
4. Roleplay Prompting - assigning a persona/role to the agent
